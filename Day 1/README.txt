What is this project about? 

This project is a "Band Name Generator" which takes 2 inputs from the user 
in the data-type of String as the city name and a pet name. It then adds them together to generate a band name. 

What did I learn from this project? 

1) Variables
2) Print Function
3) Input Function 
4) Strings
5) Math Operator: Addition
